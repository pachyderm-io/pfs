## pachctl list secret

Return information about all secrets in a given kubernetes namespace.

### Synopsis

Return information about all secrets in a given kubernetes namespace.

```
pachctl list secret [flags]
```

### Options

```
  -h, --help              Display help information.
  -n, --namespace         The Kubernetes namespace of secret (default is "default" namespace).
